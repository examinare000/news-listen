## 優先順位ルール

- **番号が大きいファイルが高優先度**
- 矛盾時は番号が大きいファイルを優先
- 同一レイヤー内では番号順に適用
- エージェント専用ルールは該当エージェントが優先

| マーク | 意味 |
|---|---|
| 🔴 | 絶対遵守必須 |
| 🟠 | 高優先度 |
| 🟡 | 中優先度 |

## 運用の要点

- 変更は小さく段階的に。危険操作は事前に目的・影響・ロールバックを宣言
- 作業前に短いプランを提示し、進捗に応じて更新
- 変更終了時は「変更ファイル一覧」「検証方法」「次の一手」を簡潔に提示

## 注意事項

- 全ファイル UTF-8 / LF 前提
- 編集時は文字コード・改行コードに注意
